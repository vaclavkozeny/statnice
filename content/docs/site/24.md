CSS – vlastnosti, hodnoty, dědění, kaskádování. Blokový model CSS

## CSS
- Cascading Style Sheets
- odděluje vzhled od struktury (HTML)
- připojení k HTML:
    - `<link rel="stylesheet" href="style.css">` – externí soubor
    - `<style>` tag v `<head>` – interní
    - `style=""` atribut přímo na elementu – inline 💀
### Vlastnosti a hodnoty
- každé pravidlo má **selektor** a **deklarace** (vlastnost: hodnota)
```css
p { 
	color: red; 
	font-size: 16px;
}
```

### Selektory
- `p` – element
- `.trida` – třída
- `#id` – ID
- `p.trida` – element s třídou
- `div > p` – přímý potomek
- `div p` – libovolný potomek
- `a:hover` – pseudotřída (stav elementu)
- `p::first-line` – pseudoelement (část elementu)
### Časté vlastnosti
- text: `color`, `font-size`, `font-weight`, `font-family`, `text-align`, `line-height`
- pozadí: `background-color`, `background-image`
- rozměry: `width`, `height`, `min-width`, `max-height`
- zobrazení: `display`, `visibility`, `opacity`
- pozicování: `position`, `top`, `left`, `z-index`
### Dědění
- **dědí se** – typicky textové vlastnosti
	- `color`, `font-size`, `font-family`, `line-height`, `text-align`
- **nedědí se** – typicky layoutové vlastnosti
	- `margin`, `padding`, `border`, `width`, `height`, `background`
### Kaskádování
- podle priority

| Priorita | Source type        | Popis                                |
| -------- | ------------------ | ------------------------------------ |
| 1        | Importance         | !important anotace, lepší nepoužívat |
| 2        | Inline             | inline style, lepší nepoužívat       |
| 5        | Selector specifity | selektory                            |
| 6        | Rule order         | poslední selektor vyhraje            |

### Blokový model
```
┌─────────────────────────────┐
│          margin             │
│   ┌─────────────────────┐   │
│   │       border        │   │
│   │  ┌───────────────┐  │   │
│   │  │    padding    │  │   │
│   │  │  ┌─────────┐  │  │   │
│   │  │  │ content │  │  │   │
│   │  │  └─────────┘  │  │   │
│   │  └───────────────┘  │   │
│   └─────────────────────┘   │
└─────────────────────────────┘
```
- **content** – samotný obsah (text, obrázek)
- **padding** – vnitřní odsazení mezi obsahem a borderem, přebírá barvu pozadí
- **border** – rámeček kolem paddingu
- **margin** – vnější odsazení od ostatních elementů, vždy průhledný
### Display a pozicování
**display** – určuje jak se element chová v layoutu:
- `block` – zabere celou šířku, začíná na novém řádku (`div`, `p`, `h1`)
- `inline` – zabere jen tolik místa kolik potřebuje, nevytváří nový řádek (`span`, `a`)
- `inline-block` – jako inline, ale lze nastavit width/height
- `flex` – flexbox layout, moderní způsob rozmístění prvků
- `grid` – grid layout, dvourozměrný
- `none` – element se nezobrazí a nezabírá místo
**position:**
- `static` – výchozí, element je v normálním toku
- `relative` – posunutý relativně od své původní pozice, ostatní elementy ho stále berou v potaz
- `absolute` – vyřazen z toku, pozicován relativně k nejbližšímu rodiči s `position` ≠ `static`
- `fixed` – vyřazen z toku, pozicován relativně k viewportu (zůstane na místě při scrollování)
- `sticky` – kombinace relative a fixed, přilepí se při scrollování
### CSS preprocesory
- nástroje které rozšiřují CSS o programovací konstrukce
- píšeš v jejich vlastní syntaxi a oni to zkompilují do běžného CSS které prohlížeč rozumí
- větší projekty ve vanilla CSS jsou těžko udržovatelné – preprocesory přinášejí strukturu
#### SCSS
```css
$primary: #333; 
nav { 
	background: $primary; 
	a { 
		color: white; 
		&:hover { 
			color: lightgray;
		} 
	} 
}
```
