## SQL
- Structured Query Language
- standardizovaný jazyk ANSI/ISO
- case insensitive
- neprocedurálni jazyk
- různé dialekty
- vždy obsahuje
	- DDL
	- DML
	- DCL
	- TCL
	- jazyk pro triggery
### DDL
- Data Definition Language
- definice databázového schématu 
- **CREATE TABLE**
	- vytvoří schéma tabulky (relace)
	- název tabulky
	- definované sloupce
	- integritní omezení nad sloupcem
	- integritní omezení nad tabulkou
	- **IO**
		- NOT NULL
		- UNIQUE
		- PRIMARY KEY
		- REFERNCES
		- CHECK - IO zadané logickým výrazem
		- DEFAULT
```SQL
CREATE TABLE Dodavatel (
	cisdod int NOT NULL PRIMARY KEY,
	jmeno nvarchar(50),
	adresa nvarchar(50),
	mesto nvarchar(50)
)
```
- **ALTER TABLE**
	- změní již vytvořenou tabulku
	- přidání/odebrání sloupce
	- změna default hodnoty
	- přidání/odebrání IO
```sql
ALTER TABLE Zamestnanci ADD Plat DECIMAL(10, 2) NOT NULL DEFAULT 0.00;

ALTER TABLE Zamestnanci ALTER COLUMN Oddeleni NVARCHAR(100);
```
- **DROP**
	- vymaže obsah tabulky a schéma tabulky ze systémového katalogu
- **TRUNCATE TABLE**
	- vymaže pouze obsah tabulky
### DML
- Data Manipulation Language
- manipulace s daty
- dotazování **SELECT**
- **INSERT**
	- vložení 1 řádky do pojmenované tabulky
```sql
INSERT INTO Studenti (sID, sJmeno, prumer, typSkoly, mesto) 
VALUES (6, 'Erik', 2.4, 'gym', 'Liberec')
```
- **UPDATE**
	- modifikace dat v DB
	- hodnoty i podmínka můžou být složené (vnořené dotazy, ...)
```sql
UPDATE Studenti 
SET mesto = 'Reichenberg' 
WHERE mesto = 'Liberec'
```
- **DELETE**
	- mazání dat z DB
	- vymazání specifické hodnoty = UPDATE na NULL
```sql
DELETE FROM Studenti 
WHERE sJmeno = 'Erika' OR sJmeno = 'Filip'
```

### SELECT
- výběr dat z DB
- Pořadí vyhodnocení: `FROM` → `WHERE` → `GROUP BY` → `HAVING` → `SELECT` → `ORDER BY`
#### Projekce
- výběr sloupců
```sql
SELECT jmeno, plat 
FROM Zamestnanci
```
#### Selekce
- výběr řádků pomocí WHERE
```sql
SELECT * 
FROM Zamestnanci 
WHERE plat > 30000
```
#### WHERE
- nalezení řádků s podmínkou
- filtrace před agregací
- možnost řetězit **AND**, **OR**, **NOT** 
- =, <, >, <=, >=, <> nebo != (nerovná se)
#### ORDER BY
- třídění
- defaultně se netřídí
- default pro třídění je **asc** (vzestupně)
- **ASC** / **DESC** (sestupně)
#### Agregace
- `COUNT(*)` – počet řádků
- `SUM(sloupec)` – součet
- `AVG(sloupec)` – průměr
- `MIN(sloupec)` / `MAX(sloupec)` – minimum / maximum
#### GROUP BY
- rozdělí řádky do skupin, agregace se počítá za každou skupinu zvlášť
```sql
SELECT oddeleni, AVG(plat) 
FROM Zamestnanci 
GROUP BY oddeleni
```
#### HAVING
- filtruje skupiny po agregaci
```sql
SELECT oddeleni, AVG(plat) 
FROM Zamestnanci 
GROUP BY oddeleni
HAVING AVG(plat) > 30000
```

#### JOIN
- **INNER JOIN** - vrátí pouze řádky, kde existuje shoda v obou tabulkách
- **LEFT JOIN** - vrátí všechny řádky z levé tabulky, u řádků bez shody vpravo doplní NULL
- **RIGHT JOIN** – opak LEFT JOIN
- **FULL OUTER JOIN** – všechny řádky z obou tabulek, kde není shoda doplní NULL
#### Vnoření
- ve WHERE
```sql
SELECT jmeno FROM Student 
WHERE id IN (SELECT student_id FROM Zapisy WHERE kurz = 'MAT')
```
- **ve FROM**
	- **subquery jako dočasná tabulka**
```sql
SELECT avg_plat FROM (
    SELECT oddeleni, AVG(plat) AS avg_plat 
    FROM Zamestnanci GROUP BY oddeleni
) AS prehled
WHERE avg_plat > 30000
```
#### LIKE
- porovnání textu se vzorem
```sql
WHERE jmeno LIKE 'No%'   -- začíná na No
WHERE jmeno LIKE '_ová'  -- jeden libovolný znak + ová
```
#### BETWEEN
```sql
WHERE plat BETWEEN 20000 AND 40000
```
#### IS
- pro testování NULL hodnoty
```sql
WHERE plat IS NULL
```
