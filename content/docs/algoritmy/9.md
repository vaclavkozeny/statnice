Číselné soustavy a převody mezi nimi. Způsoby kódování čísel s pevnou a s pohyblivou řádovou tečkou. Kódování záporných čísel

## Číselné soustavy
### DEC
- poziční číselná soustava se základem 10
- čísla 0 - 9
- přesný zápis celého čísla
- záporné čísla mají *-*
- s použitím desetinné značky lze zapsat libovolné reálné číslo
### BIN
- poziční číselná soustava se základem 2
- čísla 0 a 1
- binární číslo
- **každá číslice tedy odpovídá _n_-té mocnině čísla dvě, kde _n_ je pozice dané číslice**
- př: 214 -> 11010110
- LSB (nejméně signifikantní, pravá číslice) / MSB (nejvíce signifikantní, levá číslice)
### HEX
- poziční číselná soustava o základu 16
- znaky 0-9, A, B, C, D, E, F
- využití informatika (adresy pamětí)
- př: D6 -> 1101 0110
- 4 bity (BIN) na znak
### OCT
- bizarní číselná soustava
- poziční číselná soustava o základu 8
- čísla 0-7
- 3 bity (BIN) na znak
## Převody
#### DEC -> BIN
- dělení 2
- zbytky v opačném pořadí -> výsledek
```
9/2 = 4 | 1
4/2 = 2 | 0
2/2 = 1 | 0
1/2 = 0 | 1

9 DEC = 1001 BIN
```
#### BIN -> DEC
- součet 2^n (n je pozice číslice)
```
1001 -> 1*2^3 + 1*2^0 = 9
```
#### BIN -> HEX
- 4 znaky BIN = 1 znak HEX
```
1011 1001 = B9
```
#### HEX -> BIN
- obráceně
```
AB = 1010 1011
```
#### BIN -> OCT
- 3 znaky BIN = 1 znak OCT
```
1010 1011 = 253
```
#### OCT -> BIN
- zase obráceně
#### DEC -> HEX, OCT
- nejjednodušší převod přes BIN (DEC -> BIN -> HEX)
- opačně taky
## Řádová čárka
### Fixed point
- pevně daný rozsah celé a desetinné části
- omezená rozsah a přesnost
- 8bit -> 4 a 4
	- 0101 1100 -> 5.75 (desetinná část se mocní na zápornou mocninu pozice, začíná se od 1)
### Float
- standard
- 32bit
- 1b (sing), 8b (exponent), 23b (mantisa)

## Záporná čísla
### Sign-Magnitude
- MSB značí znaménko
	- 0 -> **+**
	- 1 -> **-**
- snížení rozsahu na polovinu
- problém 2 nul
- `+9 = 0000 1001`, `-9 = 1000 1001`
### Dvojkový doplněk
- nejpoužívanější způsob v praxi
- záporné číslo = **inverze všech bitů + 1**
```
+9 = 0000 1001
inverze  = 1111 0110
         +         1
-9 =       1111 0111
```
### Aditivní kód
- přičte se pevná konstanta
	- obvykle 2^(n-1)
- používá se hlavně u **pohyblivé řádové tečky**
